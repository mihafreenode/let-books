# Estructura de la base terminológica

## Resumen

La base terminológica de Let Books debería estructurarse para que los términos preferidos, las alternativas y las notas específicas por idioma puedan revisarse y reutilizarse de forma coherente.

## Campos sugeridos

- `term_id`
- dominio
- término preferido por locale
- alternativas por locale
- términos prohibidos por locale
- notas y ejemplos
