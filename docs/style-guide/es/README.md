# Guía de estilo

## Resumen

`docs/style-guide/` define cómo debe escribirse, localizarse, revisarse y publicarse el contenido de conocimiento de LetBooks.

## Propósito

Esta área mantiene coherente la plataforma de conocimiento a medida que crece entre idiomas y tipos de contenido.

## Estructura

- `terminology/` para el control terminológico
- `publishing/` para reglas de flujo de trabajo y herramientas
- `localization/` para reglas de traducción
- `writing/` para reglas de tono y estructura

## Temas relacionados

- Estándares editoriales
- Calidad de la localización
- Trazabilidad
