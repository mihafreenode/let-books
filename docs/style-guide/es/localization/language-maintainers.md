# Responsables de idioma

## Resumen

Los responsables de idioma ayudan a preservar la coherencia, resolver cuestiones terminológicas y coordinar la revisión para una locale determinada.

## Responsabilidades

- revisar decisiones terminológicas
- vigilar problemas de ortografía y Unicode
- coordinar preguntas de contribuyentes
- señalar formulaciones sensibles o problemas de escritura
