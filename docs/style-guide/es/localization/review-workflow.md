# Flujo de revisión

## Resumen

La revisión de localización debería pasar de la generación de borradores a una revisión humana cada vez más fiable.

## Etapas recomendadas

1. borrador de traducción
2. validación automatizada
3. revisión terminológica
4. revisión de accesibilidad
5. revisión por hablante nativo
6. revisión profesional opcional

## Requisitos para registros de revisión de IA

Cuando un revisor corrige texto generado por IA, la corrección debe documentarse en un registro reutilizable.

Campos obligatorios:

- texto original
- texto corregido
- categoría de error
- hipótesis de causa raíz
- razonamiento del revisor

Esto mantiene reutilizable el trabajo de revisión para gobernanza, entrenamiento, calibración de QA y evaluación de modelos.
