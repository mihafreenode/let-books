# Resolución de disputas

## Resumen

Los desacuerdos de traducción deben resolverse mediante razonamiento documentado y no mediante sobrescrituras silenciosas.

## Proceso

1. identificar el término o la formulación en disputa
2. comparar el significado de origen y el uso en la lengua de destino
3. comprobar las notas existentes del glosario
4. registrar la decisión y la justificación
