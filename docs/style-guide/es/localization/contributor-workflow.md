# Flujo de trabajo de contribuyentes

## Resumen

Este flujo de trabajo es para traductores, revisores, bibliotecarios, educadores y otros contribuyentes que no forman parte del código central.

## Pasos

1. elegir una página o artículo
2. comprobar la fuente canónica en inglés
3. revisar las notas terminológicas
4. enviar la traducción o los cambios de revisión
5. registrar la madurez y el estado de revisión
