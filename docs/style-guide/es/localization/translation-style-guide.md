# Guía de estilo de traducción

## Resumen

Esta guía define las expectativas básicas para la traducción del contenido de conocimiento de Let Books.

## Reglas

- preservar el significado antes del pulido estilístico
- preferir una redacción clara y didáctica
- mantener estable la terminología entre páginas relacionadas
- preservar la ortografía nativa y los caracteres Unicode
- no simplificar silenciosamente las distinciones ligadas al sistema de escritura

## Notas para la revisión asistida por IA

- no aprobar un borrador solo porque el significado general sea comprensible
- revisar con cuidado las construcciones modales porque a menudo se traducen de forma demasiado literal
- revisar el lenguaje de políticas y especificaciones del producto para un registro apropiado del dominio
- registrar ejemplos representativos de antes y después para entrenamiento futuro y referencia de QA

## Categorías de ejemplo a vigilar

- gramática
- modalidad
- terminología
- registro
- traducción literal
- pérdida de contexto
