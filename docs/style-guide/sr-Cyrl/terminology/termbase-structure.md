# Структура терминолошке базе

## Сажетак

Let Books терминолошка база треба да буде структурисана тако да преферирани термини, алтернативе и језички специфичне белешке могу доследно да се прегледају и поново користе.

## Предложена поља

- `term_id`
- домен
- преферирани термин по локализацији
- алтернативе по локализацији
- забрањени термини по локализацији
- белешке и примери
