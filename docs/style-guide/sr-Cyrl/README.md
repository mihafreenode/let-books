# Стилски водич

## Сажетак

`docs/style-guide/` дефинише како LetBooks садржај знања треба да буде написан, локализован, прегледан и објављен.

## Сврха

Ова област одржава платформу знања кохерентном како расте кроз језике и типове садржаја.

## Структура

- `terminology/` за управљање терминологијом
- `publishing/` за правила тока рада и алата
- `localization/` за правила превођења
- `writing/` за правила тона и структуре

## Повезане теме

- Уреднички стандарди
- Квалитет локализације
- Следљивост
