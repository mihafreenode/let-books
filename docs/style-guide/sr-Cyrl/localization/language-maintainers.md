# Одржаваоци језика

## Сажетак

Одржаваоци језика помажу да се очува доследност, реше терминолошка питања и координира преглед за одређену локализацију.

## Одговорности

- преглед терминолошких одлука
- праћење правописних и Unicode проблема
- координација питања сарадника
- означавање осетљивог језика или проблема са писмом
