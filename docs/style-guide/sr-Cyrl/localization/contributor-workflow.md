# Ток рада сарадника

## Сажетак

Овај ток рада намењен је преводиоцима, рецензентима, библиотекарима, едукаторима и другим сарадницима који нису део основног развојног тима.

## Кораци

1. изаберите страницу или чланак
2. проверите канонски енглески извор
3. прегледајте терминолошке белешке
4. пошаљите превод или измене прегледа
5. забележите ниво зрелости и статус прегледа
