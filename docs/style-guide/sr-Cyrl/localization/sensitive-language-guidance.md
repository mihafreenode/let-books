# Смернице за осетљив језик

## Сажетак

Неки језички избори дотичу идентитет, историју и регионалне разлике. Пројекат треба да остане пун поштовања и усмерен на имплементацију.

## Правила

- избегавајте запаљиво формулисање
- признајте разлике без заузимања страна у неповезаним споровима
- дајте предност јасноћи, приступачности и документованом образложењу
