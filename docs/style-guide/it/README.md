# Guida di stile

## Riepilogo

`docs/style-guide/` definisce come i contenuti di conoscenza di LetBooks devono essere scritti, localizzati, revisionati e pubblicati.

## Scopo

Quest'area mantiene coerente la piattaforma della conoscenza mentre cresce tra lingue e tipi di contenuto.

## Struttura

- `terminology/` per il controllo terminologico
- `publishing/` per le regole di workflow e tooling
- `localization/` per le regole di traduzione
- `writing/` per le regole di tono e struttura
