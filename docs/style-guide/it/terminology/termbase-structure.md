# Struttura della termbase

## Riepilogo

La termbase di Let Books dovrebbe essere strutturata in modo che termini preferiti, alternative e note specifiche per lingua possano essere revisionati e riutilizzati in modo coerente.

## Campi suggeriti

- `term_id`
- dominio
- termine preferito per locale
- alternative per locale
- termini vietati per locale
- note ed esempi
