# Flusso di lavoro dei contributori

## Riepilogo

Questo flusso di lavoro è per traduttori, revisori, bibliotecari, educatori e altri contributori non del codice core.

## Passi

1. scegliere una pagina o un articolo
2. controllare la fonte canonica inglese
3. rivedere le note terminologiche
4. inviare la traduzione o le modifiche di revisione
5. registrare maturità e stato della revisione
