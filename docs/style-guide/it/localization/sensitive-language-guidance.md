# Linee guida per il linguaggio sensibile

## Riepilogo

Alcune scelte linguistiche toccano identità, storia e variazione regionale. Il progetto dovrebbe rimanere rispettoso e focalizzato sull'implementazione.

## Regole

- evitare formulazioni infiammatorie
- riconoscere le differenze senza schierarsi in dispute non correlate
- privilegiare chiarezza, accessibilità e motivazione documentata
