# Responsabili linguistici

## Riepilogo

I responsabili linguistici aiutano a preservare la coerenza, risolvere questioni terminologiche e coordinare la revisione per una determinata locale.

## Responsabilità

- rivedere le decisioni terminologiche
- controllare problemi di ortografia e Unicode
- coordinare le domande dei contributori
- segnalare formulazioni sensibili o problemi di script
