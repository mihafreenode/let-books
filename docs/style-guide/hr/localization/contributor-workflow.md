# Tijek rada suradnika

## Sažetak

Ovaj tijek rada namijenjen je prevoditeljima, pregledavateljima, knjižničarima, edukatorima i drugim suradnicima koji nisu dio jezgrene razvojne ekipe.

## Koraci

1. odaberite stranicu ili članak
2. provjerite kanonski engleski izvor
3. pregledajte terminološke bilješke
4. pošaljite prijevod ili izmjene pregleda
5. zabilježite razinu zrelosti i status pregleda
