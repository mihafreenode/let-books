# Održavatelji jezika

## Sažetak

Održavatelji jezika pomažu očuvati dosljednost, rješavati terminološka pitanja i koordinirati pregled za određenu lokalizaciju.

## Odgovornosti

- pregled terminoloških odluka
- praćenje pravopisnih i Unicode problema
- koordinacija pitanja suradnika
- označavanje osjetljivog rječnika ili problema s pismom
