# Tok rada saradnika

## Sažetak

Ovaj tok rada namenjen je prevodiocima, recenzentima, bibliotekarima, edukatorima i drugim saradnicima koji nisu deo osnovnog razvojnog tima.

## Koraci

1. izaberite stranicu ili članak
2. proverite kanonski engleski izvor
3. pregledajte terminološke beleške
4. pošaljite prevod ili izmene pregleda
5. zabeležite nivo zrelosti i status pregleda
