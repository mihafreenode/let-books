# Stilhandbuch

## Zusammenfassung

`docs/style-guide/` definiert, wie LetBooks-Wissensinhalte geschrieben, lokalisiert, geprüft und veröffentlicht werden sollen.

## Zweck

Dieser Bereich hält die Wissensplattform konsistent, während sie über Sprachen und Inhaltstypen hinweg wächst.

## Struktur

- `terminology/` für Terminologiekontrolle
- `publishing/` für Regeln zu Workflow und Tooling
- `localization/` für Übersetzungsregeln
- `writing/` für Regeln zu Ton und Struktur

## Verwandte Themen

- Redaktionsstandards
- Lokalisierungsqualität
- Rückverfolgbarkeit
