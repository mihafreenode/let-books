# Workflow für Mitwirkende

## Zusammenfassung

Dieser Workflow ist für Übersetzer, Prüfer, Bibliothekare, Lehrende und andere nicht zum Kerncode gehörende Mitwirkende gedacht.

## Schritte

1. eine Seite oder einen Artikel auswählen
2. die kanonische englische Quelle prüfen
3. Terminologiehinweise prüfen
4. Übersetzung oder Prüfänderungen einreichen
5. Reifegrad und Prüfstatus dokumentieren
