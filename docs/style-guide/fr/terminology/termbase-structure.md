# Structure de la base terminologique

## Résumé

La base terminologique Let Books doit être structurée de façon à ce que les termes préférés, les alternatives et les notes spécifiques à la langue puissent être revus et réutilisés de manière cohérente.

## Champs suggérés

- `term_id`
- domaine
- terme préféré par locale
- alternatives par locale
- termes interdits par locale
- notes et exemples
