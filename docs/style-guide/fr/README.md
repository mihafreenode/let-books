# Guide de style

## Résumé

`docs/style-guide/` définit comment le contenu de connaissance LetBooks doit être rédigé, localisé, relu et publié.

## Objectif

Cette zone maintient la cohérence de la plateforme de connaissance à mesure qu'elle s'étend à plusieurs langues et types de contenu.

## Structure

- `terminology/` pour le contrôle terminologique
- `publishing/` pour les règles de flux de travail et d'outillage
- `localization/` pour les règles de traduction
- `writing/` pour les règles de ton et de structure
