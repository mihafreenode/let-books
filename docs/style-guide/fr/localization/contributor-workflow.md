# Flux de travail des contributeurs

## Résumé

Ce flux de travail est destiné aux traducteurs, réviseurs, bibliothécaires, éducateurs et autres contributeurs hors code cœur.

## Étapes

1. choisir une page ou un article
2. vérifier la source canonique anglaise
3. revoir les notes terminologiques
4. soumettre la traduction ou les changements de révision
5. enregistrer la maturité et l'état de la révision
