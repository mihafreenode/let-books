# Responsables de langue

## Résumé

Les responsables de langue aident à préserver la cohérence, résoudre les questions terminologiques et coordonner la révision pour une locale donnée.

## Responsabilités

- réviser les décisions terminologiques
- surveiller les problèmes d'orthographe et d'Unicode
- coordonner les questions des contributeurs
- signaler les formulations sensibles ou les problèmes d'écriture
