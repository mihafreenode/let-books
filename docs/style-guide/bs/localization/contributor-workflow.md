# Tok rada saradnika

## Sažetak

Ovaj tok rada namijenjen je prevodiocima, recenzentima, bibliotekarima, edukatorima i drugim saradnicima koji nisu dio osnovnog razvojnog tima.

## Koraci

1. odaberite stranicu ili članak
2. provjerite kanonski engleski izvor
3. pregledajte terminološke bilješke
4. pošaljite prijevod ili izmjene pregleda
5. zabilježite nivo zrelosti i status pregleda
