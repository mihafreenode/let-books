# Održavaoci jezika

## Sažetak

Održavaoci jezika pomažu da se očuva dosljednost, riješe terminološka pitanja i koordinira pregled za određenu lokalizaciju.

## Odgovornosti

- pregled terminoloških odluka
- praćenje pravopisnih i Unicode problema
- koordinacija pitanja saradnika
- označavanje osjetljivog jezika ili problema s pismom
