# Skrbniki jezikov

## Povzetek

Skrbniki jezikov pomagajo ohranjati doslednost, reševati terminološka vprašanja in usklajevati preglede za posamezno lokalizacijo.

## Odgovornosti

- pregled terminoloških odločitev
- spremljanje težav z ortografijo in Unicode
- usklajevanje vprašanj sodelujočih
- označevanje občutljivega ubesedovanja ali težav s pisavo
