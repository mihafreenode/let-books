# Potek dela sodelujočih

## Povzetek

Ta potek dela je namenjen prevajalcem, pregledovalcem, knjižničarjem, izobraževalcem in drugim sodelujočim, ki niso del osrednje razvojne ekipe.

## Koraki

1. izberite stran ali članek
2. preverite kanonični angleški vir
3. preglejte terminološke opombe
4. oddajte prevod ali spremembe pregleda
5. zabeležite zrelost in stanje pregleda
