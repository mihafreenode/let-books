# Терминологија

## Резиме

`docs/style-guide/terminology/` ќе ја содржи проектната терминолошка база.

## Цел

Да се одржува клучната LetBooks терминологија доследна низ UI, документација, статии, водичи за учење и преводи.

## Структура

- преферирани термини
- алтернативни термини
- забранети термини
- јазично-специфични белешки
- групирање по домен
- управување со глосар
- структура на терминолошка база

## Поврзани теми

- Локализација
- Граф на знаење
- Уредувачка доследност
