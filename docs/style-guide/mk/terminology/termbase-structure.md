# Структура на терминолошка база

## Резиме

Терминолошката база на Let Books треба да биде структурирана така што преферираните термини, алтернативите и јазично-специфичните белешки може доследно да се прегледуваат и повторно да се користат.

## Предложени полиња

- `term_id`
- домен
- префериран термин по локализација
- алтернативи по локализација
- забранети термини по локализација
- белешки и примери
