# Стилски водич

## Резиме

`docs/style-guide/` дефинира како LetBooks содржината на знаење треба да биде напишана, локализирана, прегледана и објавена.

## Цел

Оваа област ја одржува платформата за знаење кохерентна додека расте низ јазици и видови содржина.

## Структура

- `terminology/` за управување со терминологија
- `publishing/` за правила за работни текови и алатки
- `localization/` за правила за преведување
- `writing/` за правила за тон и структура

## Поврзани теми

- Уредувачки стандарди
- Квалитет на локализација
- Следливост
