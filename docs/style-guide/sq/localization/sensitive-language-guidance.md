# Udhëzime për gjuhë të ndjeshme

## Përmbledhje

Disa zgjedhje gjuhësore prekin identitetin, historinë dhe ndryshimet rajonale. Projekti duhet të mbetet respektues dhe i përqendruar te zbatimi.

## Rregulla

- shmangni formulime nxitëse
- pranoni dallimet pa mbajtur anë në mosmarrëveshje të palidhura
- jepni përparësi qartësisë, aksesueshmërisë dhe arsyetimit të dokumentuar
