# Како од продуктна спецификација да се направи имплементациски план

## Резиме

Овој водич објаснува како продуктната спецификација да се претвори во конкретен имплементациски план без губење на целта, границите и клучните правила.

## Јадро на пристапот

Издвои:

- продуктна цел
- кориснички тек
- граници на обемот
- неоспорливи правила
- резервно однесување
- критериуми за прифаќање

## Три слоја

1. слој на тек
2. системски слој
3. слој на испорака

## Контрола

- дали планот го зачувува текот
- дали вклучува документација и валидација
- дали друг соработник може да го следи без погодување
