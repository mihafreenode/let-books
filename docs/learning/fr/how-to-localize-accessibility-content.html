Comment localiser le contenu d'accessibilité

Ressources d'apprentissage Français

Ce guide explique comment localiser le texte alternatif, les légendes, les étiquettes de formulaire, les métadonnées ARIA et d'autres textes critiques pour l'accessibilité.

Liste de contrôle

  • localiser ensemble le texte visible et non visible de l’interface
  • garder les métadonnées linguistiques exactes
  • préserver la clarté d’un langage simple
  • lorsque c’est pratique, tester avec des technologies d’assistance
