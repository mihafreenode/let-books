Comment exécuter le QA de localisation dans le CI

Ressources d'apprentissage Français

Ce guide décrit comment transformer des attentes récurrentes de relecture de localisation en contrôles CI répétables.

Sujets : CI/CD

Commencer par des contrôles stables

  • fichiers manquants
  • traductions manquantes
  • liens cassés
  • incohérences de placeholders
  • métadonnées mal formées

Ajouter plus tard des contrôles plus difficiles

  • vérification orthographique dans de nombreuses langues
  • tests de base d’accessibilité
  • cohérence des captures d’écran
  • parité des diagrammes
