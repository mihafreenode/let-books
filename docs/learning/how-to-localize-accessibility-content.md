# How to Localize Accessibility Content

## Summary

This guide explains how to localize alt text, captions, form labels, ARIA metadata, and other accessibility-critical text.

## Checklist

- localize visible and non-visible interface text together
- keep language metadata accurate
- preserve plain-language clarity
- test with assistive technology where practical
