# Kako izgraditi lokalizacioni pipeline

## Sažetak

Ovaj vodič pokazuje kako izgraditi praktičan lokalizacioni pipeline koji obuhvata pisanje izvora, izradu prevoda, QA, pregled i objavljivanje.

## Tok rada

1. napiši čist kanonski izvor
2. definiši terminologiju i ciljne jezike
3. generiši ili uvezi nacrte prevoda
4. pokreni validaciju i proveru pravopisa
5. pregledaj tekst pristupačnosti i netekstualne resurse
6. po potrebi objavi sa oznakama zrelosti

## Uobičajene greške

- prevođenje pre nego što je terminologija stabilna
- zanemarivanje snimaka ekrana i dijagrama
- tretiranje mašinskog izlaza kao konačnog
