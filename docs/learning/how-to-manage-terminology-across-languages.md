# How to Manage Terminology Across Languages

## Summary

This guide explains how to keep a multilingual termbase stable across documentation, UI text, and educational content.

## Practical steps

- define preferred terms
- record alternatives and forbidden terms
- keep language-specific notes
- review reused technical vocabulary regularly
