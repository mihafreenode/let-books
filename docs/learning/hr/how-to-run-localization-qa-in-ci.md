# Kako provoditi lokalizacijski QA u CI-ju

## Sažetak

Ovaj vodič pokazuje kako ponavljajuća očekivanja pri pregledu lokalizacije pretvoriti u ponovljive CI provjere.

## Počni sa stabilnim provjerama

- datoteke koje nedostaju
- prijevodi koji nedostaju
- neispravne poveznice
- nepodudaranja u rezerviranom tekstu
- neispravni metapodaci

## Zahtjevnije provjere dodaj kasnije

- provjera pravopisa na više jezika
- osnovni testovi pristupačnosti
- dosljednost snimki zaslona
- podudarnost dijagrama
