# Kako izgraditi lokalizacijski pipeline

## Sažetak

Ovaj vodič pokazuje kako izgraditi praktičan lokalizacijski pipeline koji obuhvaća pisanje izvora, izradu prijevoda, QA, pregled i objavu.

## Tijek rada

1. napiši čist kanonski izvor
2. definiraj terminologiju i ciljne jezike
3. generiraj ili uvezi nacrte prijevoda
4. pokreni validaciju i provjeru pravopisa
5. pregledaj tekst pristupačnosti i netekstualne resurse
6. po potrebi objavi s oznakama zrelosti

## Uobičajene pogreške

- prevođenje prije nego što je terminologija stabilna
- zanemarivanje snimki zaslona i dijagrama
- tretiranje strojnog izlaza kao konačnog
