# Kako lokalizirati sadržaj pristupačnosti

## Sažetak

Ovaj vodič objašnjava kako lokalizirati alternativni tekst, natpise, oznake obrazaca, ARIA metapodatke i drugi tekst ključan za pristupačnost.

## Kontrolni popis

- zajedno lokaliziraj vidljivi i nevidljivi tekst sučelja
- održavaj jezične metapodatke točnima
- sačuvaj jasnoću jednostavnog jezika
- gdje je praktično, testiraj s pomoćnom tehnologijom
