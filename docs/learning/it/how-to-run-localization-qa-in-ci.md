# Come eseguire QA di localizzazione in CI

## Riepilogo

Questa guida descrive come trasformare aspettative ricorrenti di revisione della localizzazione in controlli CI ripetibili.

## Iniziare con controlli stabili

- file mancanti
- traduzioni mancanti
- link interrotti
- incongruenze dei placeholder
- metadati malformati

## Aggiungere controlli più difficili in seguito

- controllo ortografico in molte lingue
- test basilari di accessibilità
- coerenza degli screenshot
- parità dei diagrammi
