# Learning

## Summary

`docs/learning/` will hold structured educational material.

## Purpose

Use this area for illustrated study guides, lecture material, explainers, and future book-like teaching content.

## Structure

- concept guides
- workflow tutorials
- lecture notes
- visual explainers
- `index.html` per locale directory — generated by `tools/generate-index-pages.mjs`; lists all learning guides for that locale

## Frontmatter

Every learning guide Markdown file has YAML frontmatter with:
- `title` — the guide's H1 title
- `summary` — a one-sentence description (from the `## Summary` section)
- `topics` — an array of topic identifiers for cross-referencing with blog and wiki content

Frontmatter is added by `tools/add-md-frontmatter.mjs`.

## Topic Navigation

- Topic identifiers are shared across blog, learning, and wiki content to enable future cross-linking.
- Blog article index pages group content by these topics.
