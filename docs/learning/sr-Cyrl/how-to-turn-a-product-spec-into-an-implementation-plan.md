# Како Од Производне Спецификације Направити Имплементациони План

## Сажетак

Овај водич објашњава како производну спецификацију претворити у конкретан имплементациони план без губитка сврхе, граница и кључних правила тока рада.

## Језгро приступа

Пре разбијања спецификације у задатке издвоји:

- производну сврху
- кориснички ток
- границе обима
- неупитна правила
- резервно понашање
- критеријуме прихватања

## Три слоја плана

1. слој тока
2. системски слој
3. слој испоруке

## Контролна питања

- да ли план чува кориснички ток
- да ли укључује документацију и валидацију
- да ли други сарадник може да га спроведе без нагађања
