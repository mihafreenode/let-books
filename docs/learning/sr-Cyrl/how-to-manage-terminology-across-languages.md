# Како управљати терминологијом кроз језике

## Сажетак

Овај водич објашњава како одржати вишејезичну терминолошку базу стабилном кроз документацију, текст интерфејса и образовни садржај.

## Практични кораци

- дефиниши префериране појмове
- забележи алтернативе и забрањене појмове
- води белешке специфичне за језик
- редовно прегледај поново коришћен технички вокабулар
