# Kako zgraditi lokalizacijsko cevovodno verigo

## Povzetek

Ta vodič prikazuje, kako vzpostaviti praktično lokalizacijsko cevovodno verigo, ki pokriva pisanje izvornega besedila, pripravo prevodov, zagotavljanje kakovosti, pregled in objavo.

## Potek dela

1. napiši čist kanonični vir
2. določi terminologijo in ciljne jezike
3. ustvari ali uvozi osnutke prevodov
4. zaženi validacijo in črkovalni pregled
5. preglej besedilo za dostopnost in netekstovne gradnike
6. po potrebi objavi z oznakami zrelosti

## Pogoste napake

- prevajanje, preden je terminologija stabilna
- zanemarjanje zaslonskih posnetkov in diagramov
- obravnavanje strojnega izhoda kot končne različice
