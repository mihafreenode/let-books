# Kako upravljati terminologijo med jeziki

## Povzetek

Ta vodič pojasnjuje, kako ohraniti večjezično terminološko bazo stabilno v dokumentaciji, besedilu uporabniškega vmesnika in izobraževalni vsebini.

## Praktični koraki

- določi prednostne izraze
- zabeleži alternative in prepovedane izraze
- vodi jezikovno specifične opombe
- redno pregleduj ponovno uporabljeno tehnično besedišče
