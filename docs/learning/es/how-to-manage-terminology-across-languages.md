# Cómo gestionar la terminología entre idiomas

## Resumen

Esta guía explica cómo mantener estable una base terminológica multilingüe entre documentación, texto de la interfaz y contenido educativo.

## Pasos prácticos

- definir los términos preferidos
- registrar alternativas y términos prohibidos
- mantener notas específicas por idioma
- revisar regularmente el vocabulario técnico reutilizado
