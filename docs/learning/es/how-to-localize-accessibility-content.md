# Cómo localizar contenido de accesibilidad

## Resumen

Esta guía explica cómo localizar texto alternativo, leyendas, etiquetas de formularios, metadatos ARIA y otros textos críticos para la accesibilidad.

## Lista de comprobación

- localizar juntos el texto visible y no visible de la interfaz
- mantener exactos los metadatos de idioma
- preservar la claridad del lenguaje sencillo
- cuando sea práctico, probar con tecnología de asistencia
