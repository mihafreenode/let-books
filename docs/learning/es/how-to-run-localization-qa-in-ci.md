# Cómo ejecutar QA de localización en CI

## Resumen

Esta guía describe cómo convertir expectativas recurrentes de revisión de localización en comprobaciones repetibles de CI.

## Empezar con comprobaciones estables

- archivos faltantes
- traducciones faltantes
- enlaces rotos
- desajustes de placeholders
- metadatos mal formados

## Añadir después comprobaciones más exigentes

- corrección ortográfica en varios idiomas
- pruebas básicas de accesibilidad
- coherencia de capturas de pantalla
- paridad de diagramas
