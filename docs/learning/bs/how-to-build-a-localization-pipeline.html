Kako izgraditi lokalizacijski pipeline

Materijali za učenje Bosanski

Ovaj vodič pokazuje kako izgraditi praktičan lokalizacijski pipeline koji obuhvata pisanje izvora, izradu prevoda, QA, pregled i objavu.

Tok rada

  1. napiši čist kanonski izvor
  2. definiši terminologiju i ciljne jezike
  3. generiši ili uvezi nacrte prevoda
  4. pokreni validaciju i provjeru pravopisa
  5. pregledaj tekst pristupačnosti i netekstualne resurse
  6. po potrebi objavi s oznakama zrelosti

Uobičajene greške

  • prevođenje prije nego što je terminologija stabilna
  • zanemarivanje snimaka ekrana i dijagrama
  • tretiranje mašinskog izlaza kao konačnog
