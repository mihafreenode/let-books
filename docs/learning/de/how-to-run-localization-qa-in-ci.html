Wie man Lokalisierungs-QA in CI ausführt

Lernmaterial Deutsch

Dieser Leitfaden beschreibt, wie wiederkehrende Erwartungen aus dem Lokalisierungsreview in wiederholbare CI-Prüfungen überführt werden.

Themen: CI/CD

Mit stabilen Prüfungen beginnen

  • fehlende Dateien
  • fehlende Übersetzungen
  • kaputte Links
  • Platzhalter-Abweichungen
  • fehlerhafte Metadaten

Schwierigere Prüfungen später ergänzen

  • Rechtschreibprüfung in vielen Sprachen
  • einfache Barrierefreiheits-Tests
  • Screenshot-Konsistenz
  • Diagramm-Parität
