Early Alpha Notice

Let Books is currently an early alpha and concept-validation project.

This repository already includes a public documentation site and a browser-only local demo. Some visuals and workflow examples illustrate the intended direction of the project rather than a finished production system.

Any libraries, universities, archives, nonprofits, public institutions, companies, cloud providers, or external systems mentioned in the documentation are referenced only as conceptual examples or possible future integration scenarios.
