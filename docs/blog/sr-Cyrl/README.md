# Serbian Cyrillic Blog

## Summary

This directory will hold Serbian Cyrillic article translations.

## Purpose

Treat Serbian Cyrillic as its own supported publishing locale rather than a script conversion of another source.

## Structure

- one translation per approved article
- same `article_id` as English
- explicit cross-links to other language variants
