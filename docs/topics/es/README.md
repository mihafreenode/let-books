# Temas

`docs/topics/` contiene páginas conceptuales breves que se sitúan entre las entradas del blog, la referencia del wiki y las guías de learning.

Usa Temas para páginas de orientación que primero expliquen una idea en términos neutrales y luego la conecten con Let Books cuando resulte práctico.

Categorías actuales:

- Desarrollo guiado por especificaciones
- Localización a escala
- Metadatos de libros y bibliotecas
