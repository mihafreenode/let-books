# Теме

`docs/topics/` садржи сажете концептуалне странице које стоје између блог објава, wiki референци и водича за учење.

Теме користи за оријентационе странице које најпре неутрално објашњавају неку идеју, а затим је, где је то практично, повезују са Let Books.

Тренутне категорије:

- Развој вођен спецификацијом
- Локализација у великом обиму
- Књижни и библиотечки метаподаци
