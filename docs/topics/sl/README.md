# Teme

`docs/topics/` vsebuje jedrnate konceptualne strani, ki stojijo med blog objavami, referenčnimi wiki stranmi in učnimi vodniki.

Teme uporabi za orientacijske strani, ki najprej nevtralno razložijo idejo, nato pa jo tam, kjer je to praktično, povežejo z Let Books.

Trenutne kategorije:

- Razvoj, voden s specifikacijo
- Lokalizacija v velikem obsegu
- Knjižni in knjižnični metapodatki
