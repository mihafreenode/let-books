# Themen

`docs/topics/` enthält kurze konzeptionelle Seiten, die zwischen Blogbeiträgen, Wiki-Referenz und Learning-Guides stehen.

Verwende Themen für Orientierungsseiten, die zuerst eine Idee in neutralen Begriffen erklären und sie dann, wo sinnvoll, mit Let Books verknüpfen.

Aktuelle Kategorien:

- Spezifikationsgesteuerte Entwicklung
- Lokalisierung im großen Maßstab
- Buch- und Bibliotheksmetadaten
