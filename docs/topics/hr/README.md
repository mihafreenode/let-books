# Teme

`docs/topics/` sadrži sažete konceptualne stranice koje stoje između blog objava, wiki referenci i vodiča za učenje.

Teme koristite za orijentacijske stranice koje najprije neutralno objašnjavaju neku ideju, a zatim je, gdje je to praktično, povezuju s Let Books.

Trenutačne kategorije:

- Razvoj vođen specifikacijom
- Lokalizacija u velikom opsegu
- Knjižni i knjižnični metapodaci
