# Temi

`docs/topics/` contiene brevi pagine concettuali che si collocano tra i post del blog, la documentazione di riferimento del wiki e le guide di learning.

Usa i Temi per pagine di orientamento che spiegano prima un'idea in termini neutrali e poi la collegano a Let Books quando è utile.

Categorie attuali:

- Sviluppo guidato dalle specifiche
- Localizzazione su larga scala
- Metadati di libri e biblioteche
