# Temat

`docs/topics/` mban faqe konceptuale të përmbledhura që qëndrojnë midis postimeve në blog, referencës së wiki-t dhe udhëzuesve mësimorë.

Përdorni Temat për faqe orientuese që shpjegojnë fillimisht një ide në terma neutralë dhe pastaj e lidhin atë me Let Books aty ku është praktike.

Kategoritë aktuale:

- Zhvillimi i Udhëhequr nga Specifikimi
- Lokalizimi në Shkallë
- Metadata e Librit dhe e Bibliotekës
