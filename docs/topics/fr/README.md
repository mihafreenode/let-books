# Thèmes

`docs/topics/` contient de courtes pages conceptuelles qui se situent entre les billets de blog, la référence wiki et les guides d'apprentissage.

Utilisez les Thèmes pour des pages d'orientation qui expliquent d'abord une idée en termes neutres puis la relient à Let Books lorsque c'est utile.

Catégories actuelles :

- Développement piloté par les spécifications
- Localisation à grande échelle
- Métadonnées des livres et des bibliothèques
