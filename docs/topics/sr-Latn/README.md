# Teme

`docs/topics/` sadrži sažete konceptualne stranice koje stoje između blog objava, wiki referenci i vodiča za učenje.

Teme koristi za orijentacione stranice koje najpre neutralno objašnjavaju neku ideju, a zatim je, gde je to praktično, povezuju sa Let Books.

Trenutne kategorije:

- Razvoj vođen specifikacijom
- Lokalizacija u velikom obimu
- Knjižni i bibliotečki metapodaci
