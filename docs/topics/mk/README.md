# Теми

`docs/topics/` содржи концизни концептуални страници што стојат меѓу блог објави, wiki референци и водичи за учење.

Темите користи ги за ориентациски страници што најнапред неутрално објаснуваат некоја идеја, а потоа, каде што е практично, ја поврзуваат со Let Books.

Тековни категории:

- Развој воден од спецификација
- Локализација во голем обем
- Книжни и библиотечни метаподатоци
