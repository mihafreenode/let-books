# Translation Memory and Glossaries

## Summary

Translation memory and glossary systems improve consistency, lower review cost, and preserve institutional language decisions.

## Why they matter

- repeated phrases stay stable
- smaller teams waste less effort
- technical vocabulary becomes teachable
- reviewer disagreements shrink over time
