# Non-Text Asset Localization

## Summary

Localization should also cover diagrams, screenshots, captions, SVG assets, and other visual teaching materials.

## Rules

- avoid English-only diagrams inside localized content
- prefer structured diagram sources that can be re-rendered per language
- localize captions and alt text alongside the image
- track screenshot freshness when UI text changes
