# Modelo de madurez de localización

## Resumen

El modelo de madurez de la localización define niveles de calidad por etapas para que la cobertura de la traducción pueda mejorar honestamente con el tiempo.

## Niveles

- Nivel 0: solo idioma de origen
- Nivel 1: borrador traducido automáticamente
- Nivel 2: corrector ortográfico
- Nivel 3: automatizado QA validado
- Nivel 4: comunidad revisada
- Nivel 5: hablante nativo revisado
- Nivel 6: revisado profesionalmente

## Por qué esto importa

El modelo ayuda a los proyectos a evitar la falsa elección entre paridad perfecta y ninguna localización.
