# Memorias de traducción y glosarios

## Resumen

Las memorias de traducción y los glosarios mejoran la coherencia, reducen el trabajo de revisión y preservan las decisiones lingüísticas institucionales.

## Por qué son importantes

- las frases repetidas se mantienen estables
- los equipos pequeños necesitan menos esfuerzo adicional
- el vocabulario técnico se vuelve más fácil de usar de forma coherente
- los desacuerdos entre quienes revisan se reducen con el tiempo
