# Memorias de traducción y glosarios

## Resumen

Los sistemas de memoria de traducción y glosario mejoran la coherencia, reducen el costo de revisión y preservan las decisiones lingüísticas institucionales.

## Por qué son importantes

- las frases repetidas se mantienen estables
- los equipos más pequeños desperdician menos esfuerzo
- el vocabulario técnico se vuelve enseñable
- Los desacuerdos entre los revisores se reducen con el tiempo.
