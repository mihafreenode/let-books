# Model zrelosti lokalizacije

## Sažetak

Model zrelosti lokalizacije definira stupnjevite razine kvalitete kako bi se pokrivenost prijevoda s vremenom mogla pošteno poboljšati.

## Razine

- Razina 0: samo izvorni jezik
- Razina 1: strojno preveden nacrt
- Razina 2: pravopis provjeren
- Razina 3: potvrđeno automatizirano QA
- Razina 4: pregledana zajednica
- Razina 5: pregledan izvorni govornik
- Razina 6: stručno pregledano

## Zašto je ovo važno

Model pomaže projektima da izbjegnu lažni izbor između savršenog pariteta i nikakve lokalizacije.
