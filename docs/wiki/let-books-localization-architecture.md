# Let Books Localization Architecture

## Summary

This page summarizes how Let Books organizes multilingual docs, UI strings, generators, validation, and script-aware content policy.

## Current layers

- multilingual static docs under `docs/`
- localized static-demo UI strings under `static-demo/locales/`
- article, wiki, and learning generators under `tools/`
- validation scripts and workflows in `tools/` and `.github/workflows/`
