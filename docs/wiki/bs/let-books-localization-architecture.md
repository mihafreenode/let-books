# Let Books Arhitektura lokalizacije

## Rezime

Ova stranica rezimira kako Let Books organizira višejezične dokumente, UI nizove, generatore, validaciju i politiku sadržaja svjesnog skripte.

## Trenutni slojevi

- višejezični statički dokumenti ispod `docs/`
- lokalizirani statički demo UI nizovi ispod `static-demo/locales/`
- generatori članaka, wiki i učenja ispod `tools/`
- validacijske skripte i tokovi rada u `tools/` i `.github/workflows/`
