# Lokalizacijski model zrelosti

## Rezime

Model zrelosti lokalizacije definira etapne nivoe kvaliteta tako da se pokrivenost prijevoda može pošteno poboljšati tokom vremena.

## Nivoi

- Nivo 0: samo izvorni jezik
- Nivo 1: mašinski preveden nacrt
- Nivo 2: provjereno pravopis
- Nivo 3: automatizirano QA potvrđeno
- Nivo 4: zajednica pregledana
- Nivo 5: pregledan izvorni govornik
- Nivo 6: profesionalno pregledan

## Zašto je ovo važno

Model pomaže projektima da izbjegnu lažni izbor između savršenog pariteta i bez lokalizacije.
