# Memorija prevoda i glosari

## Rezime

Prevodilačka memorija i glosari poboljšavaju doslednost, smanjuju troškove pregleda i čuvaju institucionalne jezičke odluke.

## Zašto su važni

- ponovljene fraze ostaju stabilne
- manji timovi troše manje truda
- tehničko nazivlje postaje lakše dosledno primenjivati
- neslaganja recenzenta se vremenom smanjuju
