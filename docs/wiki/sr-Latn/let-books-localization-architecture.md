# Let Books Arhitektura lokalizacije

## Rezime

Ova stranica rezimira kako Let Books organizuje višejezične dokumente, nizove korisničkog interfejsa, generatore, validaciju i politiku sadržaja koja je svesna skripti.

## Trenutni slojevi

- višejezični statički dokumenti ispod `docs/`
- lokalizovani statički demo UI stringovi ispod `static-demo/locales/`
- generatori članaka, viki i učenja ispod `tools/`
- skripte za validaciju i tokovi rada u `tools/` i `.github/workflows/`
