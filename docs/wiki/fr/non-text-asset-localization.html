Localisation d'actifs non textuels

La localisation doit également couvrir les diagrammes, les captures d'écran, les légendes, les ressources SVG et d'autres supports pédagogiques visuels.

Sujets : Accessibility Diagrams Localization

Règles

  • évitez les diagrammes en anglais uniquement dans le contenu localisé
  • privilégier les sources de diagrammes structurés pouvant être restituées par langue
  • localiser les légendes et le texte alternatif à côté de l’image
  • suivre la fraîcheur de la capture d’écran lorsque le texte de l’interface utilisateur change
