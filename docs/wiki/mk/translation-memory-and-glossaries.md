# Преведувачка меморија и речник

## Резиме

Системите за преведувачка меморија и речник ја подобруваат конзистентноста, ги намалуваат трошоците за преглед и ги зачувуваат институционалните јазични одлуки.

## Зошто тие се важни

- повторените фрази остануваат стабилни
- помалите тимови трошат помалку труд
- техничкиот вокабулар станува подучлив
- несогласувањата на рецензентите се намалуваат со текот на времето
