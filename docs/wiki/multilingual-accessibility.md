# Multilingual Accessibility

## Summary

Multilingual accessibility covers localized language metadata, assistive-technology support, accessible text alternatives, and script-correct presentation across languages.

## Requirements

- localize ARIA labels and form labels
- localize alt text and captions
- set correct page and content language metadata
- preserve readable typography across scripts
- test dynamic announcements in more than one language where practical
